# send_line_rust
RustでLINEのブロードキャスト


## Setup

```bash
# 1. Clone This Repo
git clone https://github.com/Yokohide0317/send_line_rust.git
cd send_line_rust/

# 2. Set Env for LINE Access Token
export LINE_ACCESS_TOKEN="Your LINE Bot's Access Token"
# or put in ~/.bashrc

# 3. Build
cargo build

```

send_line_rust 処理が終わりました 確認よろ
